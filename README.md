# Mina zkApp: Snarkyjs Img

This template uses TypeScript.

## Pull submodules

```sh
git submodule update --init --recursive
```

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
